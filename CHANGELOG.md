# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [1.0.9] - 2026-01-16

### Changed
- CLI `init` and `update` commands now always delete existing `.agent` folder before installing

## [1.0.8] - 2026-01-15

### Changed
- Refactored CLI for better modularity and maintainability

## [1.0.0] - 2026-01-10

### Added
- Initial release
- CLI tool with `init`, `update`, `status` commands
- 41 domain-specific skills
- 16 specialized AI agents
- 11 workflow definitions
- Auto `.gitignore` update feature
